We've been tracking visitor attendance over the last few months, and the numbers are exciting! As we approach the warmer months, we expect even more activity.

### Monthly Visitor Statistics

Below is a chart showing the trend in visitors to Kordiak Park over the past five months.

<div class="bg-white p-4 rounded-xl shadow-inner mb-6">
    <canvas id="visitorChart" width="400" height="200"></canvas>
</div>

<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
<script>
    setTimeout(function() {
        const ctx = document.getElementById('visitorChart').getContext('2d');
        new Chart(ctx, {
            type: 'line',
            data: {
                labels: ['Jan', 'Feb', 'Mar', 'Apr', 'May'],
                datasets: [{
                    label: 'Number of Visitors',
                    data: [120, 190, 300, 550, 800],
                    borderColor: 'rgb(34, 197, 94)',
                    backgroundColor: 'rgba(34, 197, 94, 0.5)',
                    tension: 0.4
                }]
            },
            options: {
                responsive: true,
                scales: {
                    y: { beginAtZero: true }
                }
            }
        });
    }, 100);
</script>

As you can see, the upward trend is very clear. We recommend visiting on weekday mornings if you prefer quieter trails!
